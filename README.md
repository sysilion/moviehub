# MovieHub - 롯데시네마 굿즈 재고 추적 시스템

실시간으로 롯데시네마의 이벤트 굿즈(아트카드, 뱃지 등) 재고를 추적하고 시각화하는 웹 서비스 및 스케줄러입니다.

## 주요 기능

### 1. 지능형 재고 수집 (Collectors)
*   **자동 탐색**: 신규 이벤트를 분석하여 최적의 굿즈 ID(GiftID)를 자동으로 추정 및 매칭합니다.
*   **정밀 스캔**: 추정된 ID 주변부를 정방향/역방향으로 스캔하여 정확한 데이터 소스를 찾아냅니다.
*   **유연한 필터링**: 키워드 매칭(포함/제외)을 통해 추적 대상을 정교하게 선별합니다.

### 2. 고도화된 스케줄러 (Scheduler)
*   **개별 이벤트 추적**: 각 이벤트별로 독립적인 스케줄링 작업을 생성하여 격리된 환경에서 업데이트를 진행합니다.
*   **정밀 업데이트**: 09:00 ~ 24:00 사이, 5~10분 간격의 초 단위 랜덤 주기로 재고를 갱신합니다.
*   **자동 생명주기 관리**: 이벤트 종료나 재고 소진 시 추적 목록에서 자동으로 제거되어 자원을 효율적으로 관리합니다.

### 3. 직관적인 웹 대시보드 (Web UI)
*   **실시간 현황**: 대시보드에서 모든 활성 이벤트의 총 재고 상태를 한눈에 확인할 수 있습니다.
*   **사용자 친화적 표기**: 수량에 따라 '50개 이상', '개 이하', '소진'으로 직관적으로 표시합니다.
*   **변동 이력 시각화**: 특정 지점을 클릭하면 시간대별 수량 변화를 **라인 차트**로 확인할 수 있습니다.
*   **즉시 업데이트**: 사용자의 요청에 따라 즉시 최신 재고를 가져오는 백그라운드 갱신 기능을 지원합니다.

## 기술 스택
*   **Backend**: Python, FastAPI, SQLAlchemy
*   **Frontend**: Jinja2 Templates, Bootstrap 5, Chart.js
*   **Scheduler**: APScheduler
*   **Database**: SQLite (기본값)

## 설치 및 실행

### 패키지 설치
```bash
pip install -r requirements.txt
```

### 서비스 실행
```bash
python main.py
```
*   웹 서버: `http://localhost:8000`
*   스케줄러: 백그라운드에서 자동 실행

## 데이터베이스 구조
*   `events`: 이벤트 기본 정보 (이름, 기간, 이미지 등)
*   `inventory`: 지점별 최신 재고 현황
*   `inventory_history`: 수량 변동 시계열 데이터 (차트용)
